## Описание проекта

Проект ориентирован на автоматизированное извлечение и объединение данных о побочных эффектах препаратов. Для этого используются:
- **Парсинг PDF-инструкций ГРЛС.** Информация извлекается непосредственно из документов.
- **Обращение к внешним источникам.** В случаях, когда данные из ГРЛС недоступны или неполны, проект использует сайты [rlsnet.ru](https://rlsnet.ru) и [drugs.com](https://drugs.com) для заполнения недостающих сведений.

В результате формируются структурированные датасеты с информацией о побочных эффектах, сопровождаемые метаданными, такими как главы и ссылки на оригинальные ресурсы.

## Структура репозитория

- **make_side_effect_dataset**  
  Сценарий, отвечающий за парсинг информации из различных источников с соблюдением приоритетности (ГРЛС → rlsnet.ru → drugs.com).

- **data**  
  Хранит ключевые данные и файлы:
  - `drugs_table4.csv` – таблица со ссылками на ГРЛС, rlsnet.ru и drugs.com.
  - `side_effects_dict_translation.json` – словарь для перевода терминов побочных эффектов.
  - `list_class_side_e_edit.txt` – список основных категорий побочных эффектов.
  - `side_dataset.json` – датасет с информацией о побочных эффектах, разделённой по главам и с ссылками на источники.
  - `sef_dataset.json` – датасет, содержащий данные по частоте возникновения побочных эффектов.

- **CustomPymorphy**  
  Пользовательские модификации для библиотеки морфологического анализа (Pymorphy).

- **compute_object_p, convertors, extract_text_from_instructions, process_yEd_graph, utils**  
  Дополнительные модули для обработки, конвертации данных, извлечения текста из инструкций и других вспомогательных операций.

## Структура репозитория polypharmacy-nikita

В репозитории находятся следующие файлы и папки:

- **CustomPymorphy/** – модуль с пользовательскими модификациями для библиотеки морфологического анализа Pymorphy.

- **compute_object_p/** – модуль для вычисления «ошибки свёртки» (convolution error) лекарственных средств на основе данных из различных источников.

- **convertors/** – набор скриптов для конвертации и преобразования данных между форматами.

- **data/** – папка для хранения ключевых данных и файлов проекта. Содержит, в частности:

    - *drugs_table4.csv* – таблица со ссылками на источники информации о препаратах (ГРЛС, rlsnet.ru, drugs.com).

    - *side_effects_dict_translation.json* – словарь для перевода терминов, связанных с побочными эффектами.

    - *list_class_side_e_edit.txt* – список основных категорий побочных эффектов.

    - *side_dataset.json* – датасет с информацией о побочных эффектах, разделённой по главам и дополненной ссылками на исходные источники.

    - *sef_dataset.json* – датасет с данными о частоте возникновения побочных эффектов.

- **doccano/** – файлы и настройки для работы с Doccano (инструментом разметки текстовых данных).

- **extract_text_from_instructions/** – скрипты для извлечения текста из инструкций к лекарственным средствам.

- **make_side_effect_dataset/** – содержит скрипты для парсинга информации о побочных эффектах из разных источников (по приоритету: ГРЛС, затем rlsnet.ru и drugs.com).

- **parse_vigiaccess/** – скрипты для получения и парсинга данных из системы VigiAccess (международной базы данных о побочных реакциях на лекарства).

- **process_yEd_graph/** – утилиты для обработки и анализа графов, созданных в графическом редакторе yEd.

- **text_corpus_addiction/** – данные и скрипты для работы с текстовым корпусом, посвящённым зависимости (addiction).

- **utils/** – утилитарный модуль с вспомогательными функциями для различных задач проекта.

- **visualization_embedding/** – скрипты и модули для визуализации эмбеддингов (векторных представлений) данных.
